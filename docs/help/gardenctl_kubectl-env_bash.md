## gardenctl kubectl-env bash

Generate a script that points KUBECONFIG to the targeted cluster for bash

### Synopsis

Generate a script that points KUBECONFIG to the targeted cluster for bash.

To load the kubectl configuration script in your current shell session:
$ eval $(kubectl-env bash)


```

```
